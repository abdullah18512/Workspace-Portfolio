# Workspace Portfolio

> A developer portfolio disguised as a VS Code workspace.

## 🔗 Links
- **Live:** Coming soon
- **GitHub:** github.com/abdullah18512

---

## 🛠️ Built With
- React + Vite
- Tailwind CSS v4
- react-markdown
- Lucide React
- react-type-animation

---

## ✨ Features
- VS Code-inspired UI with Explorer, Tabs & Editor
- Recursive file tree with expand/collapse
- PDF viewer for resume
- JSON viewer for contact info
- Interactive terminal with custom commands
- Typewriter animation on terminal outputs

---

## 💡 Why I Built This
I wanted my portfolio to stand out. Instead of a typical
scrolling website, I built a workspace that feels alive —
because I spend most of my time in VS Code anyway.
